# 📚 Đọc Truyện - App Android

## ✅ Cách BUILD APK dễ nhất (Windows)

### Yêu cầu:
- Máy tính Windows 10/11
- Kết nối Internet
- **KHÔNG cần cài gì trước**

### Các bước:

1. **Giải nén** file ZIP này ra Desktop
2. Vào thư mục `DocTruyen`
3. **Chuột phải** vào file `BUILD_APK.bat` → **"Run as administrator"**
4. Chờ script tự động cài đặt và build (~10-15 phút lần đầu)
5. File `DocTruyen.apk` sẽ xuất hiện trên Desktop
6. Gửi file APK qua **Zalo/Telegram** sang điện thoại
7. Mở file APK trên điện thoại và cài đặt

> **Lưu ý:** Lần đầu sẽ download ~700MB (JDK + Android SDK). 
> Lần sau build lại chỉ mất ~1 phút.
