# 📚 Đọc Truyện - Hướng dẫn lấy APK

## 🎯 Chỉ 3 bước — không cần cài gì!

---

## BƯỚC 1: Tạo tài khoản GitHub (miễn phí, 2 phút)

1. Vào https://github.com/signup
2. Nhập email, mật khẩu, tên người dùng
3. Xác nhận email là xong

---

## BƯỚC 2: Upload source code lên GitHub

1. Đăng nhập vào https://github.com
2. Nhấn nút **"+"** (góc trên bên phải) → **"New repository"**
3. Đặt tên: `doctruyen` → nhấn **"Create repository"**
4. Trên trang repository vừa tạo, tìm dòng **"uploading an existing file"** → nhấn vào
5. **Kéo thả TOÀN BỘ các file và thư mục** trong ZIP này vào trang web
   (Kéo cả thư mục `.github`, `app`, `gradle`, và tất cả file `.gradle`, `.bat`, `.md`)
6. Nhấn **"Commit changes"** (màu xanh)

---

## BƯỚC 3: Tải APK về

1. Trong repository, nhấn tab **"Actions"**
2. Thấy dòng **"Build APK"** đang chạy (biểu tượng ⏳ vàng) — chờ ~5 phút
3. Khi thấy dấu ✅ xanh → nhấn vào
4. Cuộn xuống phần **"Artifacts"**
5. Nhấn **"DocTruyen-APK"** để tải về
6. Giải nén file .zip tải về → có file **DocTruyen.apk**

---

## 📱 Cài APK lên điện thoại

1. Gửi file `DocTruyen.apk` qua **Zalo** hoặc **Telegram** sang điện thoại
2. Mở file trên điện thoại
3. Nếu hỏi "Cho phép cài từ nguồn không rõ" → nhấn **Cho phép**
4. Nhấn **Cài đặt** → **Xong!**

---

## ❓ Câu hỏi thường gặp

**Actions bị lỗi đỏ?**
→ Nhấn vào dòng lỗi → xem log → thường do upload thiếu file

**Không thấy tab Actions?**
→ Vào Settings → Actions → General → bật "Allow all actions"

**Điện thoại không cài được?**
→ Vào Cài đặt → Bảo mật → bật "Cài ứng dụng từ nguồn lạ"
